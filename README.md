# PyQuest Academy

เว็บสอน Python สำหรับผู้เริ่มต้นแบบเกมด่าน โทนมืดสบายตา รันได้เป็น static site และนำขึ้น GitHub Pages ได้ฟรี

## จุดเด่น

- หน้าแรกเป็นแผนที่เลือกด่านแบบเกม และหน้าบทเรียนเป็นหน้าฝึกเฉพาะกิจแบบไม่รก
- สอนก่อนแล้วค่อยให้พิมพ์จริง พร้อมบอกคำตอบที่ถูกก่อนในด่านต้น ๆ
- มีระบบหัวใจ XP แพ้/ชนะ คิวทบทวน และปลดล็อกด่านใหม่ต่อเนื่อง
- ใช้ Gemini API keys ได้หลายอัน พร้อมสลับโมเดลและ fallback อัตโนมัติ
- หากยังไม่มี API key จะมีครูสำรองในเครื่องสร้างด่านพื้นฐานให้ก่อน
- ใช้ `localStorage` เก็บโปรไฟล์ ความคืบหน้า ด่านที่สร้างแล้ว และการตั้งค่าบนเครื่องผู้ใช้

## วิธีรันในเครื่อง

ใช้เซิร์ฟเวอร์ไฟล์ธรรมดาได้เลย เช่น

```powershell
python -m http.server 4173
```

จากนั้นเปิด `http://localhost:4173`
